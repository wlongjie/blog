RPC解释

  • RPC：Remote Procedure Call(远程过程调用)
  • 客户端获取服务端的服务(不同的计算机)

RPC所需要的技术

  • 反射技术：客户端给服务端发送代表接口名的字符串，服务端需要通过字符串解析出该字符串代表的接口的一切信息
  • socket：客户端与服务端交互(传输信息)

  • 动态代理：服务端需要根据客户端的不同请求，返回不同的接口类型，客户端需要接受到不同的接口类型
